# 🦞 OpenClaw for Google IDX (Lite)

这是去掉了 PM2 管理器的轻量版 OpenClaw 模版。

## 🚀 快速开始
1. 在 IDX 中导入本仓库。
2. 等待 `setup.sh` 运行结束。
3. 在终端输入 `source ~/.bashrc`。
4. 运行 `openclaw onboard` 完成初次设置。
5. **运行服务**：直接输入 `openclaw gateway` 即可。

## 💡 提示
因为去掉了 PM2，如果你关闭了运行 Gateway 的终端窗口，服务就会停止。
建议开一个专属的终端窗口来运行 `openclaw gateway`。
